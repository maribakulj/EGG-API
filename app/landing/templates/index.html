Museum — adds a museum sub-block (inventory number, artist, medium, dimensions, acquisition date, current location) and enables an IIIF manifest passthrough. Fits Axiell, TMS, Micromusée, Mobydoc.

{{ t("landing.not.heading") }}

{% if lang == "fr" %}
  • Pas un site grand public ni une interface de découverte — c'est l'API JSON sous le front-end que vous utilisez déjà (ou que votre prestataire construira par-dessus).
  • Pas un moteur d'indexation — il se place devant votre moteur de recherche (Elasticsearch, OpenSearch) et en expose une vue propre.
  • Pas un SGC — votre SIGB / DAMS / CMS d'archives reste la source de vérité ; EGG-API se contente de republier.
{% else %}
  • Not an end-user website or a discovery front-end — it is the JSON API underneath whichever front-end you already use (or the one a prestataire builds on top).
  • Not an indexing engine — it sits in front of your existing search backend (Elasticsearch, OpenSearch) and exposes a curated shape.
  • Not a content management system — your SIGB/DAMS/archives CMS stays the source of truth; EGG-API just republishes.
{% endif %}
